Getting your own copy

Navigate to your Gmod install location

For me its E:\SteamLibrary\steamapps\common\GarrysMod.

Then navigate into the "garrysmod" folder and then navigate into the "lua" folder

so now your file explorer should show you \steamapps\common\GarrysMod\garrysmod\lua

from here you need to place our install lua found below.

Please note that there may be some lag/throttling due to downloading 10-15 files asyncherously

When completed you should recieve a download complete code telling you to enter PZ_init into the console

Running PZ_init will activate the base loader and allow plugins to be loaded.

To run your plugin with Project Zenith do PZ_compile_url %YourURLHere.com/FancyLuaName.lua% however we would prefer to have your plugin hosted on the Github then we would simply do PZ_add_plugin "name" so itd be not only easier for everyone but simpler.

How does Project Zenith Work?

It's actually pretty simple.

Local Computer

#1. GarrysMod is running and user has just ran the installation script ingame.

GitHub Servers

#2. The client attempts to download the master files from the repository from github.

Local Computer

#3. The downloaded files have been ran and compiled in the current process GarrysMod.

Project Zenith

#4. Open the console to find information about PZ and what commands are available.


Types of codes

We use the standard HTTP status codes as a framework

1xx - Informational

100 - Continue

The server has recived the request headers and the client should proceed to send the request body.

2xx - Success

200 - Ok

Standard response for successful HTTP requests

201 - Created

The request has been fulfilled, resulting in the creation of a new resource.

202 - Accepted

The request has been accepted but hasent completed processing.

3xx - Redirect

4xx - Client Errors

400 - Bad Request

420 - Enhance Your Calm

You're being rate limited please slow down your requests

5xx - Server Error
